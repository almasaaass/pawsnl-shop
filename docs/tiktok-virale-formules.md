## De Gouden Regels

| Regel | Waarom |
|-------|--------|
| **Eerste 3 sec = hook** | 63% van virale video's haken kijkers in 3 sec |
| **15-25 seconden** | 70%+ voltooiing nodig voor virale distributie |
| **Shares > Likes** | Shares wegen 3x zwaarder in het algoritme |
| **Casual > professioneel** | Overproductie is een conversion killer |
| **3-5x per week posten** | Volume + variatie = kans op virale hit |
| **Tekst overlays altijd** | Veel kijkers kijken zonder geluid |
| **50-100 tekens caption** | 21% meer likes dan lange captions |
| **3-5 hashtags** | Mix breed (#fyp) + niche (#pawcleaner) |
| **Post di/do/vr 10-14u** | Of 18-21u CET voor avondscrollers |

## Bewezen Hook Formules (Kopieer & Plak)

### Curiosity Gap
- "Ik wou dat ik dit eerder had ontdekt..."
- "Wacht tot je het in het donker ziet..."
- "Je raadt nooit hoe schoon zijn poten werden"

### Directe Aanspreking
- "Stop met scrollen als je een hond hebt"
- "Elke avondwandelaar heeft DIT nodig"
- "Hondenbazen, luister..."

### Bold Claim
- "Dit is waarom elke hondenbaas dit nodig heeft"
- "Het beste hondenproduct dat ik ooit kocht"
- "€16,95 en mijn hond is het veiligste dier op straat"

### Emotie/Angst
- "Mijn hond was bijna ONZICHTBAAR in het donker..."
- "Je hond na een regenachtige wandeling..."

### Fout/Mistake
- "De #1 fout die hondenbazen maken na een wandeling"
- "Je hebt €0 uitgegeven aan dit product en dat is een fout"

## Workflow: Gratis Video Maken

1. **Kies een concept** hierboven
2. **Maak video** in InVideo AI (gratis, 4/week) of A2E.ai (30 credits/dag)
   - Plak het script als prompt
   - Kies 9:16 formaat
3. **Bewerk in CapCut** (gratis)
   - Voeg tekst overlays toe
   - Voeg trending sound toe (zoek in TikTok Creative Center)
   - Trim tot 15-25 sec
4. **Upload handmatig** naar TikTok
5. **Plak de URLs** in Admin → Video Pipeline → Edit
